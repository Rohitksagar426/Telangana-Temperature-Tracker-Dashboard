## Dataset Information

The dataset contains Telangana weather data from 2021–2024, organized by year.

Each yearly folder includes multiple CSV files with columns such as:
- Date
- District
- Max Temperature (°C)
- Min Temperature (°C)
- Rainfall (mm)
- Humidity (%)
- Wind Speed (kmph)

Used for building the Power BI Telangana Temperature Tracker Dashboard.
